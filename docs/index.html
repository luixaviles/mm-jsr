About M&M JSR

Mort&Mortis JS Range (M&M JSR) is library for JavaScript. It provides You with excellent solution for creating so-called range-inputs. Range input is form's field where one can choose a value from min-max range. Although HTML 5 comes with input[type="range"] its functionality lacks a lot of features. M&M JSR gives You anything You may need.

M&M JS Range was created as an alternative to popular ion.rangeSlider. Being competition for such a big brand I decided to focus on features ion.rangeSlider doesn't have: better keyboard support, support for screen-readers (which is very hard part and is being developed), lightweight, no dependencies, any number of sliders and others. See comparison...

HTML markup, basic usage, multiple sliders, labels, disabled min/max labels.

<input id="range-1-1" type="range" min="0" max="100" value="30">
<input id="range-1-2" type="range" min="0" max="100" value="30">
<input id="range-1-3" type="range" min="0" max="100" value="30">
new JSR(['#range-1-1', '#range-1-2', '#range-1-3'], {
    sliders: 3,
    values: [25, 50, 75],
    labels: {
        minMax: false
    },
    grid: false
});

Large values, limit (visible!), affix.

new JSR(['#range-2-1', '#range-2-2'], {
    sliders: 2,
    min: 10000,
    max: 20000,
    values: [15000, 17500],
    labels: {
        formatter: (value) => {
            return '$' + value.toString();
        }
    },
    limit: {
        show: true
    },
    grid: false
}).setLimit('min', 12000).setLimit('max', 16000);

Small step, single value, one side limit, grid with some color.

new JSR('#range-3', {
    sliders: 1,
    step: 0.01,
    values: [50],
    limit: {
        min: 19
    },
    labels: {
        formatter: (value) => {
            return value.toString() + ' PLN';
        }
    },
    grid: {
        color: '#999'
    }
});

Negative values.

new JSR(['#range-4-1', '#range-4-2'], {
    sliders: 2,
    step: 0.1,
    min: -200,
    max: 100,
    values: [50, 75],
    grid: false
});

Small values, events listening.

new JSR(['#range-5-1'], {
    sliders: 1,
    step: 1,
    min: 1,
    max: 3,
    values: [2],
    grid: false
}).addEventListener('update', (input, value) => {
    console.log(`Custom events test: New value set: ${input}/${value}`);
});

Disabled module (labels).

new JSR(['#range-6-1'], {
    sliders: 1,
    values: [50],
    modules: {
        labels: false,
    },
    grid: false
});

Custom formatting labels, grid (formatted with label formatter).

new JSR(['#range-7-1'], {
    sliders: 1,
    values: [1],
    min: 1483228801000,
    max: 1514678399000,
    step: 1000 * 60 * 60 * 24,
    labels: {
        formatter: function (value) {
            const date = new Date(value);
            const day = date.getUTCDate();
            const month = date.getMonth() + 1;
            const year = date.getUTCFullYear();
            return `${day}-${month}-${year}`;
        }
    },
    grid: {
        color: '#777'
    }
});
