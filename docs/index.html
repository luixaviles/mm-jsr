About M&M JSR

M&M JS Range (M&M JSR v2) provides You with excellent solution for creating so-called range-inputs.

Range input is form's field where one can choose a value from min-max range. Although HTML 5 comes with input[type="range"] its functionality lacks a lot of features. M&M JSR gives You anything You may need. For more information see README

Demo

These demos represents usage of different JSR functions and settings. See README for more details.

Feel free to play with sliders, and try to find bugs :-) If you find any, let me know!

Basic usage, multiple sliders, labels.

<div id="demo1"></div>
new JSR({
    modules: [
        new JSR.Rail(),
        new JSR.Slider(),
        new JSR.Bar(),
        new JSR.Label(),
    ],
    config: {
        min: 0,
        max: 100,
        step: 1,
        initialValues: [25,75],
        container: document.getElementById('demo1'),
    }
});

Large values, limit (visible!), affix.

new JSR({
    modules: [
        new JSR.Rail(),
        new JSR.Slider(),
        new JSR.Bar(),
        new JSR.Label({
            formatter: (v) => `$${v}`
        }),
        new JSR.Limit(),
    ],
    config: {
        min: 10000,
        max: 20000,
        step: 1,
        initialValues: [15000, 17500],
        container: document.getElementById('demo2'),
        limit: {
            min: 12000,
            max: 16000,
        }
    }
});
